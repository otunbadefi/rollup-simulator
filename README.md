Simple Rollup Simulation

This project simulates a basic rollup architecture using Solidity and Foundry.

What it demonstrates

- Separation of execution and settlement layers
- State batching concept used in rollups
- Simplified state root verification
- Basic sequencer role simulation

Why I built this

I’m learning how Ethereum scaling works under the hood.

This project is part of my transition from Web3 ecosystem/community work into protocol-level engineering and infrastructure research.

Concepts explored

- L2 execution environments
- L1 settlement verification
- State roots and hashing
- Sequencer role in rollups

Stack

- Solidity
- Foundry
